# Kubernetes Practice 🚀

Hands-on Kubernetes learning repository documenting my practical journey through core Kubernetes concepts, architecture, and production-grade workloads.

## 📌 About This Repository

This repo contains my day-by-day Kubernetes hands-on practice, YAML manifests, notes, and experiments while learning Kubernetes from fundamentals to advanced orchestration concepts.

It is part of my public DevOps learning journey where I focus on **understanding by building**, not just theory.

---

## 🧠 Topics Covered

### Day 1 – Day 9 (Kubernetes Journey)

| Day    | Topic                                   | Key Concepts Practiced                                    |
| ------ | --------------------------------------- | --------------------------------------------------------- |
| Day 0 | Kubernetes Architecture & Cluster Setup | Control Plane, Worker Nodes, kubectl, kind/minikube       |
| Day 1 | Kubernetes Manifests & First Pods       | Pod YAML, Imperative vs Declarative, Labels               |
| Day 2 | Namespaces & Deployments                | ReplicaSets, Scaling, Rolling Updates                     |
| Day 3 | Services                                | ClusterIP, NodePort, LoadBalancer, DNS                    |
| Day 4 | ConfigMaps & Secrets                    | Environment Variables, Mounted Configs, Secret Management |
| Day 5 | Persistent Volumes & PVCs               | Storage, Static/Dynamic Provisioning                      |
| Day 6 | StatefulSets                            | Stable Identity, Persistent Storage, Ordered Scaling      |
| Day 7 | Resource Limits & Probes                | Requests/Limits, OOMKilled, Liveness/Readiness/Startup    |
| Day 8 | Metrics Server & HPA                    | Metrics Server, Autoscaling, CPU-Based Scaling            |
| Day 9 | Helm                                    | Charts, Releases, Templates, Upgrade/Rollback             |
